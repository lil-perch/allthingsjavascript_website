Adapt Learning

At All Things JavaScript we have a unique interest in online learning (eLearning). For decades, eLearning tools have neglected the latest standards and internet technologies in favor of a more proprietary way of doing things. We feel that Adapt changes all that.

What is Adapt?

Adapt is a free and easy to use e-learning authoring tool that creates fully responsive, multi-device, HTML5 e-learning content using the award-winning Adapt developer framework.

The Adapt developer framework uses these JavaScript frameworks/libraries: RequireJS, Backbone.js, Handlebars, Grunt and JSON for course content and configurations.

All Things JavaScript's Involvement in Adapt

Right now at All Things JavaScript we are learning and exploring the Adapt Framework. Because online education and JavaScript are both important to us, we hope to increase our involvement in the Adapt Framework.
