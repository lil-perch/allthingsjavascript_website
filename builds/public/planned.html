Learn Modern JavaScript: Advanced Topics

Projected release date: May, 2017

This is the next course in the Learn Modern JavaScript series. Advanced Topics will build on the groundwork established in the Getting Started course. Here is a list of topics that will be covered in this course:

  • First Class Functions
  • Closures, Callbacks, IIFEs
  • this Binding
  • Various Think Like a Programmer Topics
  • Prototypes and Prototypal Inheritance
  • Constructors
  • Objects in Depth
  • Namespace/Module Pattern
  • JSON
  • Programming Paradigms
  • Functional Programming Concepts
  • OOP

Learn Modern JavaScript: Asynchronous Coding

Projected release date: July, 2017

Asynchronous code is at the heart fo the real power of JavaScript. In this course you will learn traditional methods of asynchronous coding as well as new techniques such as promises.

Learn Modern JavaScript: Regular Expressions

Projected release date: July, 2017

Regular Expressions requires a course by itself. At the conclusion of this course you will be able to use and write regular expressions in your JavaScript applications with ease.
